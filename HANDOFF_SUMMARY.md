# 🚀 Antigravity 프로젝트 인계 문서 (Handoff Summary)

이 문서는 작업을 새로운 환경(Notebook)으로 이전하여 계속 진행하기 위한 가이드입니다.

## 📦 이전해야 할 항목
1. **소스 코드 폴더**: `local-tasks/` 전체
2. **이 문서**: `HANDOFF_SUMMARY.md`

## 🛠️ 새 환경 설정 방법
1. **Node.js 확인**: 새 환경에 Node.js가 설치되어 있는지 확인하세요.
2. **의존성 설치**: 터미널에서 프로젝트 폴더로 이동 후 명령어 실행
   ```bash
   cd local-tasks
   npm install
   ```
3. **서버 실행**:
   ```bash
   npm run dev
   ```
4. **접속**: 브라우저에서 `http://localhost:3000` 열기

## ✅ 현재 구현된 기능 상태

### 1. 기본 기능
- 할일 추가/수정/삭제
- 카테고리 관리 (사이드바)
- 캘린더 뷰 (월간 일정)
- 드래그 앤 드롭 정렬

### 2. 최근 구현된 기능 (Latest Features)
- **🌙 다크 모드**: 우측 상단 토글, 모든 컴포넌트(달력, 다이얼로그 등) 완벽 지원
- **📅 오늘 필터 (Today)**: `T` 단축키 또는 헤더 버튼으로 오늘 마감 항목만 보기
- **↩️ 실행 취소 (Undo)**: 항목 삭제 시 5초간 화면 하단에 [되돌리기] 알림 표시
- **📂 아카이브**: 완료된 항목 보관소
- **⌨️ 키보드 단축키**:
    - `N`: 새 할일
    - `T`: 오늘 필터 토글
    - `/`: 검색
    - `Ctrl+Enter`: 저장 (태그 입력 중에도 작동)
- **📝 메모 확장**: 메모 입력창 확장 기능 및 전용 다이얼로그
