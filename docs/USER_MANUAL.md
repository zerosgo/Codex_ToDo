# 📖 ToDo 앱 사용 설명서

> 기능별 상세 사용 가이드

---

## 🏠 앱 화면 구성

| 영역 | 설명 |
|------|------|
| 왼쪽 사이드바 | 카테고리 목록, 뷰 전환 버튼 |
| 가운데 캘린더 | 월별 일정 표시 |
| 오른쪽 Task List | 선택한 카테고리의 할일 목록 |

---

## 📅 캘린더 사용법

### 일정 보기

- **월 이동**: 상단 `◀` `▶` 버튼 또는 `Ctrl + ←/→`
- **오늘로 이동**: `오늘` 버튼 클릭
- **일요일 숨기기/보기**: `일요일 숨기기` 버튼

### 일정 추가

1. 캘린더에서 원하는 날짜 클릭
2. 팝업에서 일정 정보 입력
3. `저장` 버튼 클릭

### 일정 수정

- **원클릭 모드**: 일정 한 번 클릭 → 상세창 열림
- **더블클릭 모드**: 일정 두 번 클릭 → 상세창 열림
- 설정에서 모드 변경 가능

### 일정 이동 (드래그 앤 드롭)

1. 일정을 마우스로 길게 클릭
2. 원하는 날짜로 끌어서 놓기
3. 자동으로 날짜 변경됨

### 일정 복사

1. `Ctrl` 키를 누른 상태로
2. 일정을 다른 날짜로 드래그
3. 복사본 생성됨

## ✅ Task List 사용법

### 할일 완료 체크

- 왼쪽 `○` 클릭 → 완료 표시
- 완료된 항목은 회색으로 표시

### 할일 상세 보기

- 할일 카드 클릭 → 상세 모달 열림
- 제목, 담당자, 메모, 태그 등 수정 가능

### 제목 빠른 수정

1. 제목 부분 더블클릭
2. 새 제목 입력
3. `Enter` 키로 저장

### 할일 삭제

1. 할일 카드에 마우스 올리기
2. 휴지통 아이콘 클릭
3. 확인 팝업에서 `삭제` 클릭

### 할일 순서 변경

1. 왼쪽 `⋮⋮` 드래그 핸들 클릭
2. 원하는 위치로 끌어서 놓기

## 🔍 스마트 검색

### 검색창 열기

- `Ctrl + /` 단축키
- 또는 검색 아이콘 클릭

### 기본 검색

검색어 입력 → 제목, 담당자, 태그, 메모에서 검색

### 자연어 검색

| 입력 예시 | 동작 |
|-----------|------|
| `오늘` | 오늘 일정만 표시 |
| `내일` | 내일 일정만 표시 |
| `이번 주` | 이번 주 일정 표시 |
| `지난주` | 지난주 일정 표시 |
| `이번 달` | 이번 달 일정 표시 |
| `지난달` | 지난달 일정 표시 |
| `지난주 회의` | 지난주 + "회의" 검색 |

### 최근 검색 기록

- 검색창 열면 최근 검색어 표시
- 클릭하면 바로 검색됨
- `지우기`로 기록 삭제

## 📎 자료 URL 관리

### URL 빠르게 추가하기

1. 웹페이지 URL 복사 (`Ctrl + C`)
2. 캘린더 또는 Task List에서 해당 일정을 `Ctrl + 클릭`
3. URL 자동 저장!

### 기존 URL이 있을 때

- `확인` → 새 URL로 덮어쓰기
- `취소` → 기존 URL 유지 + 새 URL 추가

### URL 열기

- 일정 옆 📎 클립 아이콘 클릭
- 새 탭에서 자료 페이지 열림

### 여러 URL 관리

- 한 일정에 여러 URL 저장 가능
- 클립 아이콘 옆 숫자로 URL 개수 확인

---

## 🏷️ 태그 사용법

### 태그 추가

1. 일정 상세 모달 열기
2. 태그 입력란에 태그 입력
3. `Enter` 또는 `,`로 추가

### 태그로 필터링

- 태그 클릭 → 해당 태그 포함 일정만 표시

### 태그 삭제

- 태그 옆 `×` 클릭

---

## 📋 하위 할일 (체크리스트)

### 하위 할일 추가

1. 일정 상세에서 `+ 하위 할일 추가` 클릭
2. 항목 입력 후 `Enter`

### 완료 체크

- 하위 할일 왼쪽 체크박스 클릭

### 진행률 확인

- 일정 카드에 `3/5` 형태로 표시됨
- (완료된 개수 / 전체 개수)

---

## 📂 카테고리 관리

### 카테고리 추가

1. 사이드바 하단 `+ 카테고리 추가` 클릭
2. 이름 입력 후 `Enter`

### 카테고리 색상 변경

1. 카테고리 우클릭
2. 색상 선택

### 여러 카테고리 선택 (멀티 선택)

- `Ctrl + 클릭`으로 여러 카테고리 동시 선택
- 선택된 카테고리의 모든 일정이 캘린더에 표시

## 🎨 뷰 모드 전환

### 캘린더 뷰

- 기본 뷰: 월별 일정 표시
- 왼쪽 사이드바에서 `캘린더` 클릭

### Keep 뷰

- 메모장 스타일 뷰
- 왼쪽 사이드바에서 `Keep` 클릭

### 즐겨찾기 뷰

- 즐겨찾기한 일정만 표시
- 왼쪽 사이드바에서 `즐겨찾기` 클릭

---

## ⚙️ 설정

### 설정 열기

- 캘린더 우측 상단 ⚙️ 아이콘 클릭

### 클릭 모드 설정

| 모드 | 동작 |
|------|------|
| 원클릭 | 한 번 클릭으로 상세 열기 |
| 더블클릭 | 두 번 클릭으로 상세 열기 |

### 일정 높이 조절

- `Ctrl + Shift + ↑` → 높이 증가
- `Ctrl + Shift + ↓` → 높이 감소

---

## 📊 팀 일정 보기

### 팀 일정만 보기

- `팀 일정` 버튼 클릭
- 또는 `Ctrl + M` 단축키

### 이전 보기로 돌아가기

- `이전 보기` 버튼 클릭
- 또는 `Ctrl + M` 다시 누르기

### 임원 일정 필터

- `임원 일정` 버튼 클릭
- 대표/사업부/센터 관련 일정만 표시

## ❓ 문제 해결

### Q: 데이터가 사라졌어요!

브라우저 캐시/쿠키를 삭제하면 데이터도 지워집니다.
→ 중요 데이터는 정기적으로 백업하세요.

### Q: 일정이 안 보여요!

1. 올바른 카테고리가 선택되어 있는지 확인
2. 팀 일정 필터가 켜져 있는지 확인
3. 올바른 월을 보고 있는지 확인

### Q: 드래그 앤 드롭이 안 돼요!

일정의 왼쪽 `⋮⋮` 핸들을 사용해서 드래그하세요.

### Q: 단축키가 안 먹어요!

1. 입력란에 커서가 있으면 단축키가 안 됩니다
2. 브라우저 단축키와 충돌할 수 있습니다 (Edge 권장)
